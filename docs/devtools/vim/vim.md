# vim

Typescriptのテキストエディタとして使いたい。

## 最低限の使い方

### ファイルを開く

```bash
vim hogefile
```

### コマンドモードと編集モード

最初はコマンドモードで開かれる。
文字編集のためには編集モードにする必要あり。
コマンドモードは半角でおこなうこと。

###  編集する

[i] キーで編集モードにできる。

### 編集モードからコマンドモードに戻る

編集モードで[esc]キー

### 保存して終了する

コマンドモードで[:][w][q]

### 保存せずに終了する

コマンドモードで[:][!][q]
