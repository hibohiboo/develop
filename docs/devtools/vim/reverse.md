# vim

## ファイル一覧を見たい

```
:e
```

### ファイルプレビュー

ファイルの上でp

[ctrl] + [w] [hjkh]でフォーカス移動

## タブを開く

```
:tabnew
```


|gt|次のタブへ|
|--|--|
|gT|前のタブへ|

## 画面分割

```
:split
:vsplit
```

## 入力補完

[ctrl]+[p] 次の候補  
[ctrl]+[n] 前の候補
