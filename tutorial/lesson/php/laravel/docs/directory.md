## ディレクトリ構成

- bootstrap
  - アプリケーションで最初に実行される処理。オートローディング設定。
- config
  - アプリケーションの設定
- database
  - データベース関連。マイグレーション・初期投入データ
- public
  - ドキュメントルート
- resources/
  - view のテンプレート・less, sass
- routes
  - ルート定義
- storage
  - ファイルの出力先。ログ・ファイルキャッシュ・テンプレートファイルなど
- tests
  - テストコード
- vendoer
  - composer でダウンロードされるファイル群
