## 拡張機能のインストール

.vscode extension 配下にフォルダを配置

| OS      | フォルダの場所                   |
| ------- | -------------------------------- |
| Windows | %USERPROFILE%\.vscode\extensions |
| macOS   | \$HOME/.vscode/extensions        |
| Linux   | \$HOME/.vscode/extensions        |
